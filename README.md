# datamind-ai
Desenvolver um assistente de análise de dados baseado em IA que apoie Tech Analysts na compreensão de datasets, bases de dados, folhas de cálculo e queries SQL, reduzindo o esforço de análise manual.  A plataforma deve gerar automaticamente documentação, identificar problemas de qualidade de dados, explicar relações entre tabelas e apoiar a criação de SQL.
